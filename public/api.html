Make API calls by using the following template:
/specs/%Specification%

Replace %Specification% with the name of a Boeing BAC specification or a search string.

For example,
/specs/BAC5307
will return the following matching JSON objects:
[
{
"_id":"60c2b5af69370525fcfee73b",
"specification":"BAC5307-1",
"revision":"E",
"title":"PART MARKING OF MACHINE READABLE INFORMATION WITH LABELS",
"date":"06-APR-2018"
},
{
"_id":"60c2b5af69370525fcfee73a",
"specification":"BAC5307",
"revision":"AA",
"title":"PART MARKING",
"date":"22-APR-2020" } ]

Note that search strings only match with the specification number (BAC####) at this time. Calls such as /specs/5307 will not return any results.
