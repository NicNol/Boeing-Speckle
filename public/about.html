About

Boeing Speckle ("Boeing Spec. Cull") is an API to get the latest revision level of Boeing specifications. Boeing specification revision levels are updated monthly and published in one of many PDFs on the D1-4426 website. Boeing Speckle scrapes these PDFs and provides an API for returning information about all or specific specifications in JSON format.

Boeing Speckle currently lists all BAC prefixed documents (no departures).

Boeing Speckle is an open source project. View the source code on Github.

Boeing Speckle is an open source project developed by Nic Nolan.
